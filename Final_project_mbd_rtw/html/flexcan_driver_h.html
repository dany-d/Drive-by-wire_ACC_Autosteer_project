732  /*!



733   * @brief FlexCAN Rx FIFO field configuration



734   *



735   * @note The number of elements in the ID filter table is defined by the



736   *       following formula:



737   *       - for format A: the number of Rx FIFO ID filters



738   *       - for format B: twice the number of Rx FIFO ID filters



739   *       - for format C: four times the number of Rx FIFO ID filters



740   *       The user must provide the exact number of elements in order to avoid



741   *       any misconfiguration.



742   *



743   *       Each element in the ID filter table specifies an ID to be used as



744   *       acceptance criteria for the FIFO as follows:



745   *       - for format A: In the standard frame format, bits 10 to 0 of the ID



746   *         are used for frame identification. In the extended frame format, bits



747   *         28 to 0 are used.



748   *       - for format B: In the standard frame format, bits 10 to 0 of the ID



749   *         are used for frame identification. In the extended frame format, only



750   *         the 14 most significant bits (28 to 15) of the ID are compared to the



751   *         14 most significant bits (28 to 15) of the received ID.



752   *       - for format C: In both standard and extended frame formats, only the 8



753   *         most significant bits (7 to 0 for standard, 28 to 21 for extended) of



754   *         the ID are compared to the 8 most significant bits (7 to 0 for



755   *         standard, 28 to 21 for extended) of the received ID.



756   *



757   * @param   instance           A FlexCAN instance number



758   * @param   id_format          The format of the Rx FIFO ID Filter Table Elements



759   * @param   id_filter_table    The ID filter table elements which contain RTR



760   *                             bit, IDE bit, and Rx message ID



761   */



762  void FLEXCAN_DRV_ConfigRxFifo(



763    uint8_t instance,



764    flexcan_rx_fifo_id_element_format_t id_format,



765    const flexcan_id_table_t *id_filter_table);
